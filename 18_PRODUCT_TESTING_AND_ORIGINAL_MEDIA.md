# Task 18 - Product Testing And Original Media

## Muc tieu

Quyet dinh 1-3 san pham Tanh co the tu mua/test/chup anh, tao testing protocol don gian, va nang cap it nhat mot page bang original media/notes.

## Input

- Money pages da publish.
- `working/money_pages/evidence/product_evidence_tracker.csv`
- Budget thuc te cua owner.
- Camera/phone va kitchen setup cua owner.

## Output

- Product testing shortlist.
- Test protocol.
- Original photos/notes duoc luu trong working area hoac asset workflow phu hop.
- It nhat mot page duoc update bang hands-on notes neu testing xong.

## Product Selection Criteria

- Gia thap va de mua.
- De test trong kitchen hien co.
- Co lien quan den top money page.
- De chup anh khong can studio.
- Testing tao insight that, khong chi lap lai spec sheet.

## Suggested First Tests

- Sink organizer or sponge caddy: cheap, easy to photograph, useful for small-kitchen workflow.
- Dish rack or roll-up rack: good visual evidence but needs sink-fit photos.
- Mini/manual coffee tool: useful if owner already drinks coffee.
- Rice cooker: higher effort; only do if it will be used normally.

## Checklist

- [ ] Chon 1-3 products co budget.
- [ ] Tao test protocol cho moi product.
- [ ] Chup original photos: installed, storage, scale/measurement, cleaning parts.
- [ ] Ghi measurement notes: width/depth/height, counter/sink fit, cleaning friction.
- [ ] Ghi what surprised us / who should skip it.
- [ ] Update evidence tracker.
- [ ] Update page voi hands-on note va image source.
- [ ] Run static/content audits.

## Photo Policy

- Khong dung Amazon images.
- Tu chup thi ghi source la original.
- Neu dung manufacturer image, dam bao permission/source plan ro.
- Sau Amazon approval, Amazon assets chi dung qua hop le SiteStripe/PA API/tool duoc phep.

## Done

- It nhat mot page co original media hoac original measurement/testing notes.
- Evidence tracker cap nhat source la original.
