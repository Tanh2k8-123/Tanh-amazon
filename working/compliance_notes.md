## Must-have before affiliate links

- [ ] Public website URL is live.
- [ ] About page identifies Tanh and explains editorial approach.
- [ ] Contact page exists.
- [ ] Privacy Policy exists.
- [ ] Affiliate Disclosure page exists.
- [ ] Footer includes: `As an Amazon Associate I earn from qualifying purchases.`
- [ ] Each affiliate article has disclosure before the first affiliate link.
- [ ] No copied Amazon prices, ratings, reviews, or product images.
- [ ] Product images are owned, licensed, manufacturer-approved, or provided by an allowed Amazon mechanism.
- [ ] Affiliate links are logged and checked.

## Default disclosure block

```text
Disclosure: This post contains affiliate links. If you buy through these links, we may earn a commission at no extra cost to you. As an Amazon Associate I earn from qualifying purchases.
```

## Link-level disclosure labels

Preferred short label:

```text
Paid link
```

Alternative clear label:

```text
Amazon paid affiliate link - we may earn a commission.
```

Do not use `Amazon affiliate link` alone as the only link-level disclosure, because it may not clearly tell every reader that compensation is involved.

## Zero-budget hosting compliance note

Vercel Hobby should not be used for the production affiliate site because its free Hobby plan is non-commercial and Vercel's fair use guidelines treat affiliate-linking-as-primary-purpose as commercial usage.

Use Netlify Free or Cloudflare Pages Free for the public static site, then re-check current terms before launch.
